# Personal Web Page - Wang Chin Yen

A minimalist, high-end editorial personal portfolio website for **Wang Chin Yen**.

## Design Inspiration
This project is inspired by the **Hostinger "Presson" template**, featuring:
- Bold Brutalist Aesthetic: High-contrast layout using charcoal (#1E1E1E) and light gray (#D9D9D9).
- - Premium Typography: Uses the "Inter" font family with oversized, bold headings.
  - - Asymmetrical Layout: Modern and dynamic sectioning with significant whitespace.
    - - Live Clock: A real-time updating header clock for a professional, precise feel.
     
      - ## Project Structure
      - - index.html: Main structure and content of the single-page website.
        - - index.css: Comprehensive styling, typography, and animations.
          - - hero_background.png: Custom-generated architectural hero image.
           
            - ## Key Features
            - - Responsive Design: Optimized for both desktop and mobile viewing.
              - - Micro-animations: Subtle rotations and transitions (e.g., the sunburst icon).
                - - SEO Ready: Proper semantic HTML and meta tags included.
